# 🧸 Toy Store - Full Stack Web Application

A full-stack e-commerce web application for browsing and managing kids' toys.

## 🚀 Features

- 🛍️ Browse products (public)
- 🔍 Filter & sort products
- 🛒 Cart functionality
- 🔐 Authentication (JWT)
- 👤 User registration & login
- 🔒 Protected checkout
- 🛠️ Admin panel (CRUD for products)
- 🎨 Modern UI with Material UI

## 🏗️ Tech Stack

### Frontend
- React (Vite)
- Material UI
- React Router
- Axios

### Backend
- Node.js
- Express
- MongoDB (Mongoose)
- JWT Authentication
- bcrypt
